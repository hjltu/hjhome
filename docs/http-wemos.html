ESP8266 MQTT client ------------------------------------------------------------------------ Name: wemos2 IP address: 192.168.0.129 MAC address: B4:E6:2D:23:9B:85 MQTT Server: 192.168.0.10 ------------------------------------------------------------------------ Commands: (/wemos2/out/start) start, disconnect Commands: (/wemos2/in/command) echo, test, ip, mac, srv, info, reset, memory, state, name, ssid, pass, ,mqtt, period, http, blinds, shutters, buttons,term, count, value0,1,2,3, gap, sensor, default Commands descriptions: name - new name, ssid - new ssid, pass - new pass, mqtt - IP for server state - actual out14,out16,pwm12,pwm13 when power on http - web server blinds - set out12, out13 for blinds(somfy) buttons - set out12, out13 for buttons shutters - set out14, out16 for shutters term - anti freezing function for temp4<10℃ and rele14 term - anti overheating function for temp4>35℃ and rele16 count - counter for in0, in2 value0,1,2,3 - unsigned long 2147483647 max gap - interval 6(2)sec-30(7)min for sensors temp,hum,(an) period - send temp5,temp4,hum4, stat periodicaly sensor - send an0(A0) stat periodicaly press bt0 and bt2 for 20 sec to reset(default) default - for default network settings default settings: name=wemos, ssid=hjhome default settings: pass=pass1234, mqtt=192.168.0.10 topic: /wemos2/out/an0 payload: 1024 ------------------------------------------------------------------------ Pin I/O Topic Payload Status gpio0, D3 Button /wemos2/in/bt0 0,1,2 0 gpio0, D3 Counter /wemos2/in/value0 0-(2^32-1) 0 gpio0, D3 Counter /wemos2/in/value1 0-(2^32-1) 0 gpio2, D4 Button /wemos2/in/bt2 0,1,2 0 gpio2, D4 Counter /wemos2/in/value2 0-(2^32-1) 0 gpio2, D4 Counter /wemos2/in/value3 0-(2^32-1) 0 gpio4, D2 DHT /wemos2/in/temp4,hum4 ℃,% 0℃, 0% gpio5, D1 DS18B20 /wemos2/in/temp5 ℃ 0℃ gpio(12,13), D6,D7 Somfy /wemos2/out/somfy11 0-255 0 gpio12, D6 Pwm /wemos2/out/pwm12 0-255 111 gpio12, D6 Button /wemos2/in/bt12 0,1,2 0 gpio13, D7 Pwm /wemos2/out/pwm13 0-255 0 gpio13, D7 Button /wemos2/in/bt13 0,1,2 0 gpio14, D5 Rele /wemos2/out/rele14 0,1 1 gpio(14,16), D5,D0 Shutters /wemos2/out/shut15 0-255 0 gpio16, D0 Rele /wemos2/out/rele16 0,1 1 A0 Sensor /wemos2/in/an0 1-1024 1024 A0 Button /wemos2/in/bt17 0,1,2 0 EEPROM state / wemos2/in/state 0,1 1 EEPROM period / wemos2/in/period 0,1 0 EEPROM http / wemos2/in/http 0,1 1 EEPROM blinds / wemos2/in/blinds 0,1 0 EEPROM shutters / wemos2/in/shutters 0,1 0 EEPROM buttons / wemos2/in/buttons 0,1 0 EEPROM term / wemos2/in/term 0,1 0 EEPROM count / wemos2/in/count 0,1 0 EEPROM gap / wemos2/in/gap 0-255 5 EEPROM sensor / wemos2/in/sensor 0,1 1 EEPROM default / wemos2/in/default 0,1 0 ------------------------------------------------------------------------ hjmqtt 22-Aug-2016 -
