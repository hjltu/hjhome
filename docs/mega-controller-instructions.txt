NAME:         "mega11"
IP:           192.168.0.11
MQTT SERVER:  192.168.0.10

1. Connect to the mqtt server(mosquitto_sub -h 192.168.0.10 -t /# -v)
2. Connect the controller.

As a result, you should see this message:
/mega11/out/start start

Enter: mosquitto_pub -h 192.168.0.10 -t /mega11/in/ip -m ip
And if You receive IP address, open it in browser.
